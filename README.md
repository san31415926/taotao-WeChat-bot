# 微信群发工具

自动控制微信 PC 版，实现批量群发消息到多个微信群。支持图形界面和命令行两种方式。

> **开发：** 淘淘数码  
> **声明：** 本软件仅限于授权用户内部使用，禁止用于任何商业用途。

---

## 功能

- **批量群发** — 自动搜索群名前缀，依次发送到多个群
- **图形界面** — 参数编辑、文件勾选、运行控制、发送统计一应俱全
- **定时模式** — 设定时间自动发送，无需值守
- **速度控制** — 可调倍率，从正常到极速自由切换
- **紧急停止** — 鼠标移左上角或按停止按钮，立即中止
- **配置持久化** — 参数保存后下次启动自动加载

## 快速开始

```bash
# 安装依赖
pip install pyautogui pyperclip pygetwindow schedule

# 图形界面（推荐）
python gui.py

# 或命令行
python send_wechat.py --once
```

也可直接运行打包好的 `微信群发工具.exe`（无需 Python 环境）。
