## 功能概述

自动控制微信 PC 版，实现批量群发消息到多个微信群。

### 核心流程

1. 扫描目录下的 TXT 文件，读取内容
2. 打开微信，搜索自己并发送消息
3. 右键转发 → 搜索群名前缀 → 勾选群 → 发送
4. 自动切换到下一个前缀，重复直到全部发完

## 运行方式

### 方式一：图形界面（推荐）

双击 `微信群发工具.exe`，或运行：

```
python gui.py
```

界面包含：
- **配置参数** — 群名前缀、群数量、定时时间、速度倍率、间隔秒数、日志级别
- **文件选择** — 列出目录下所有 TXT 文件，勾选要发送的
- **操作按钮** — 立即运行 / 定时模式 / 查看统计 / 保存配置 / 停止

### 方式二：命令行

```
python send_wechat.py          # 交互菜单
python send_wechat.py --once   # 立即运行
python send_wechat.py --daemon # 定时模式
python send_wechat.py --stats  # 查看统计
python send_wechat.py --getpos # 测量鼠标坐标
```

## 配置参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `group_prefixes` | 群名前缀（如 00A001、00A002） | 00A001~00A008 |
| `groups_per_prefix` | 每个前缀有几个群 | 9 |
| `txt_files` | 要发送的 TXT 文件列表 | 苹果12~17.txt |
| `send_times` | 定时发送时间（24小时制） | 09:00, 12:00, 18:00 |
| `speed_factor` | 速度倍率（0.2=极速 0.5=2倍速 1.0=正常） | 0.5 |
| `interval_between_groups` | 前缀之间等待秒数 | 1 |
| `log_level` | 日志详细程度 | INFO |

---

## 速度倍率参考

| 倍率 | 效果 | 适用场景 |
|------|------|----------|
| 0.2 | 极速 | 网络好、微信响应快时 |
| 0.3 | 3倍速 | 大部分情况可用 |
| 0.5 | 2倍速 | **推荐**，稳定与速度平衡 |
| 1.0 | 正常速度 | 网络慢或微信卡顿时 |

## 紧急停止

两种紧急停止方式：

1. **鼠标移到屏幕左上角 (0,0)** — 脚本会在 0.1 秒内停止
2. **GUI 点"停止"按钮** — 同样立即生效

停止后需要重新点"立即运行"或"定时模式"才能继续。

## 注意事项

1. **微信必须已登录**，且窗口不被最小化到托盘
2. 第一次使用前建议用 `--getpos` 校准鼠标坐标
3. 发送内容文件编码必须是 UTF-8
4. 定时模式依赖 `schedule` 库，运行后不要关闭窗口
5. `pyautogui.FAILSAFE` 默认开启，鼠标移左上角可紧急停止
